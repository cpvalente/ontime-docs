```bash title="Lower thirds"
https://MY-IP-ADDRESS:4001/lower           
```

The `lower thirds` page is an automated view which generates lower thirds based on current runtime status. \
This workflow allows for [integration with video servers capable of rendering a webpage](/quick-tips/render-elsewhere), such as OBS Studio.
