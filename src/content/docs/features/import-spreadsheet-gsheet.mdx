You can synchronise Ontime to a Google Sheet. \
This is ideal for collaboration or extensive data manipulation.

![Spreadsheet template](../../../assets/screenshots/features/spreadsheet-template.png)
Follow [here][spreadsheet-link] to see the Google Sheet from the screenshot
To enable this feature, you will to need provide Ontime with the necessary permissions.\
From there, we can read and write to the Google Sheet spreadsheet.

:::note[Prerequisites]
To use the Google Sheet integration, you will a Google account and to the Google Console. \
The machine running Ontime must have access to the internet.
:::

<Steps>
  <ol>
    <li>
      ### Create project
      <details>
        <summary>Go to https://console.cloud.google.com/ and create a new project</summary>
        ![step-1.1](../../../assets/screenshots/features/sheet-guide/step-1.1.jpg)
        ![step-1.2](../../../assets/screenshots/features/sheet-guide/step-1.2.jpg)
        ![step-1.3](../../../assets/screenshots/features/sheet-guide/step-1.3.jpg)
      </details>

      <details>
        <summary>On the front page select the newly created project</summary>
        ![step-1.1](../../../assets/screenshots/features/sheet-guide/step-1.1.jpg)
        ![step-1.4](../../../assets/screenshots/features/sheet-guide/step-1.4.jpg)
      </details>
    </li>
    <li>
      ### Enable the Sheet API
      <details>
        <summary>Open the sidebar and click "APIs and Services"</summary>
        ![step-2.00](../../../assets/screenshots/features/sheet-guide/step-2.00.jpg)
        ![step-2.01](../../../assets/screenshots/features/sheet-guide/step-2.01.jpg)
      </details>
      <details>
        <summary>Click "Enable APIs and Services"</summary>
        ![step-2.02](../../../assets/screenshots/features/sheet-guide/step-2.02.jpg)
      </details>

      <details>
        <summary>... and search for "Sheets"</summary>
        ![step-2.2](../../../assets/screenshots/features/sheet-guide/step-2.2.jpg)
        ![step-2.3](../../../assets/screenshots/features/sheet-guide/step-2.3.jpg)
        ![step-2.4](../../../assets/screenshots/features/sheet-guide/step-2.4.jpg)
      </details>
    </li>
    <li>
      ### Setup Data Access
      <details>
        <summary>Add the necessary scopes</summary>
        Go to "Data Access" and click "Add or remove scopes"
        This could depend on your organization is set up but it should make sense by reading through the options
        ![step-3.1](../../../assets/screenshots/features/sheet-guide/step-3.1.jpg)

        Find the correct scopes by filtering for "sheets"
        ![step-3.4](../../../assets/screenshots/features/sheet-guide/step-3.4.jpg)

        Enable ".../auth/spreadsheets" and click "Update"
        ![step-3.5](../../../assets/screenshots/features/sheet-guide/step-3.5.jpg)
      </details>

      <details>
        <summary>Click save and continue</summary>
        ![step-3.6](../../../assets/screenshots/features/sheet-guide/step-3.6.jpg)
        For "Test users" add the email(s) the will be using this. \
        Save and go back to the dashboard
      </details>

      <details>
        <summary>Add test user</summary>
        Go to "Audience" and click "Add users"
        Here add the email(s) the will be using this. \
        Remember to save
        ![step-3.7](../../../assets/screenshots/features/sheet-guide/step-3.7.jpg)
      </details>
    </li>
    <li>
      ### Create credentials
      <details>
        <summary>Go to "Clients"</summary>
        ![step-4.1](../../../assets/screenshots/features/sheet-guide/step-4.1.jpg)
      </details>

      <details>
        <summary>Click "Create client"</summary>
        ![step-4.2](../../../assets/screenshots/features/sheet-guide/step-4.2.jpg)
      </details>

      <details>
        <summary>For Application type select "TVs and Limited input device", give it a name and click "create"</summary>
        ![step-4.3](../../../assets/screenshots/features/sheet-guide/step-4.3.jpg)
      </details>

      <details>
        <summary>Download the result, we are now done with the Google console</summary>
        ![step-4.5](../../../assets/screenshots/features/sheet-guide/step-4.5.jpg)
      </details>
    </li>
    <li>
      ### Connect Ontime
      <details>
        <summary>Go to the sheet sync settings</summary>
        ![step-5.1](../../../assets/screenshots/features/sheet-guide/step-5.1.jpg)
        ![step-5.2](../../../assets/screenshots/features/sheet-guide/step-5.2.jpg)
        ![step-5.3](../../../assets/screenshots/features/sheet-guide/step-5.3.jpg)
      </details>

      <details>
        <summary>Upload the file we have downloaded from Google console</summary>
        ![step-5.4](../../../assets/screenshots/features/sheet-guide/step-5.4.jpg)
      </details>

      <details>
        <summary>Get the sheet ID, paste it in Ontime and click connect</summary>
        ![step-5.5](../../../assets/screenshots/features/sheet-guide/step-5.5.jpg)
        ![step-5.6](../../../assets/screenshots/features/sheet-guide/step-5.6.jpg)
      </details>

      <details>
        <summary>Copy the code and click authenticate and follow the instructions from Google</summary>
        ![step-5.7](../../../assets/screenshots/features/sheet-guide/step-5.7.jpg)
      </details>
    </li>

  </ol>
</Steps>
