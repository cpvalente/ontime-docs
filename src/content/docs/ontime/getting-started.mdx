## What is Ontime

Ontime is a free, open-source rundown and event timer for live productions. \
It manages rundowns, scheduling, and cueing, and supports live productions, broadcasts, and events with real-time collaboration and powerful integrations.

You can run Ontime anywhere: install it on your machine, run it on a production network, or use it in the cloud.

## Installing Ontime

Ontime is available as a desktop app for Windows, macOS, and Linux, or as a Docker image you can self-host on your own server.

Download the latest release from GitHub and run the installer for your platform. \
If your operating system warns you about an unsigned app, see our [notes on unsigned apps](/additional-notes/unsigned-app/).

## What is Ontime Cloud

Ontime Cloud is the fastest way to run Ontime online, without installing or managing any infrastructure yourself. \
It runs the same Ontime software you would self-host, so your project data remains portable between Cloud and local versions.

Ontime Cloud is a good fit if you want to collaborate with your team without setting up a network, or if you need Ontime available before your production network exists.
