You can control most of the playback functions of ontime over HTTP protocol.\
This could be handy for integration with third party software, [including vMix](/additional-notes/control-from-vmix).

:::note[Notes on requests]
Any request which successfully changes the Ontime playback is responded with a `Status Message 202`.
:::

| Address                      | Payload                                       | Return                                                      | Description                                                      |
| :--------------------------- | :-------------------------------------------- | :---------------------------------------------------------- | :--------------------------------------------------------------- |
| `/playback/`                 |                                               | `roll` `play` `pause` `stop` `armed`                        | Get current playback state                                                          |
| `/playback/start`            | `{eventId: string}` \| `{eventIndex: number}` |                                                             | Start playing given event                                                         |
| `/playback/pause`            |                                               |                                                             | Pause a running event                                                          |
| `/playback/roll`             |                                               |                                                             | Start [Roll Mode](/features/roll)                                                          |
| `/playback/previous`         |                                               |                                                             | Load previous event in rundown                                                   |
| `/playback/next`             |                                               |                                                             | Load next event in rundown                                                         |
| `/playback/load`             | `{eventId: string}` \| `{eventIndex: number}` |                                                             | Load given event                                                         |
| `/playback/unload`           |                                               |                                                             | Unload (stop) currently loaded event                                                          |
| `/playback/reload`           |                                               |                                                             | Reload currently loaded event                                                          |
