# Event Properties

the allowed event properties are

| property | value type                                                                                                  |
| -------- | ----------------------------------------------------------------------------------------------------------- |
| title    | string                                                                                                      |
| note     | string (including new line)                                                                                 |
| cue      | string (try to keep this small)                                                                             |
| duration | number (in seconds)                                                                                         |
| isPublic | boolean                                                                                                     |
| skip     | boolean                                                                                                     |
| colour   | string (# hex colours or [named css colours](https://developer.mozilla.org/en-US/docs/Web/CSS/named-color)) |
| custom   | target the specific custom field with 'custom:fieldname'                                                    |

# Message Destination

the allowed message destinations are

| destinations |                                          |
| ------------ | ---------------------------------------- |
| timer        | the will be overlaid on the stage timer  |
| public       | the will be displayed on public displays |
| lower        | for use with the lower third generator   |
