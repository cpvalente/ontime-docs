The OSC API allow for low latency communication to Ontime server.

import { Tabs, TabItem } from '@astrojs/starlight/components';

<TabItem label="OSC">
| Adresse                                               | Arguments | Description                                                 | Return                            |
| ----------------------------------------------------- | --------- | ----------------------------------------------------------- | --------------------------------- |
| /ontime/version                                       |           |                                                             | v3.xx.xx                          |
| /ontime/poll                                          |           | Returns the entire runtime state                            | [runtime data](/api/runtime-data) |
| /ontime/change/`ID`/`property`                        | `value`   | Changes the `property` of the event with `ID` to be `value` | updated event                     |
| /ontime/message/[timer,public,lower,external]/text    | `value`   | Sets the text of the message controller to `value`          | updated message controller        |
| /ontime/message/[timer,public,lower,external]/visible | `value`   | Sets the visibility of the message controller to `value`    | updated message controller        |
| /ontime/message/timer/timerBlink                      | `value`   | Sets the blinking of the message controller to `value`      | updated message controller        |
| /ontime/message/timer/timerBlackout                   | `value`   | Sets the blackout of the message controller to `value`      | updated message controller        |
| /ontime/start                                         |           | Starts the first event                                      |
| /ontime/start/index                                   | `value`   | Starts the event at index `value`                           |
| /ontime/start/id                                      | `value`   | Starts the event with id `value`                            |
| /ontime/start/cue                                     | `value`   | Starts the event with cue name `value`                      |
| /ontime/load                                          |           | Starts the first event                                      |
| /ontime/load/index                                    | `value`   | Loades the event at index `value`                           |
| /ontime/load/id                                       | `value`   | Loades the event with id `value`                            |
| /ontime/load/cue                                      | `value`   | Loades the event with cue name `value`                     |
| /ontime/addtime                                       | `value`   | Adds `value` seconds to the running event                   |
| /ontime/pause                                         |           | Pauses the playback                                         |
| /ontime/stop                                          |           | Stops the playback                                          |
| /ontime/reload                                        |           | Reloads selected event                                      |
| /ontime/roll                                          |           | Sets playback mode to roll                                  |
</TabItem>
