import { Tabs, TabItem } from "@astrojs/starlight/components";

<Tabs>
<TabItem label="OSC">

All OSC messages shold start with **/ontime/**

| Adresse                                                | Description                                                                         |
| ------------------------------------------------------ | ----------------------------------------------------------------------------------- |
| /version                                               | Returns `v3.xx.xx`                                                                  |
| /poll                                                  | Returns the entire [runtime state](/api/runtime-data)                               |
| /change/`ID`/`property` `value`                        | Changes the `property` of the event with `ID` to be `value` updated event           |
| /message/[timer,public,lower,external]/text `value`    | Sets the text of the message controller to `value` updated message controller       |
| /message/[timer,public,lower,external]/visible `value` | Sets the visibility of the message controller to `value` updated message controller |
| /message/timer/timerBlink `value`                      | Sets the blinking of the message controller to `value` updated message controller   |
| /message/timer/timerBlackout `value`                   | Sets the blackout of the message controller to `value` updated message controller   |
| /start                                                 | Starts the first event                                                              |
| /start/index `value`                                   | Starts the event at index `value`                                                   |
| /start/id `value`                                      | Starts the event with id `value`                                                    |
| /start/cue `value`                                     | Starts the event with cue name `value`                                              |
| /load                                                  | Starts the first event                                                              |
| /load/index `value`                                    | Loades the event at index `value`                                                   |
| /load/id `value`                                       | Loades the event with id `value`                                                    |
| /load/cue `value`                                      | Loades the event with cue name `value`                                              |
| /addtime `value`                                       | Adds `value` seconds to the running event                                           |
| /pause                                                 | Pauses the playback                                                                 |
| /stop                                                  | Stops the playback                                                                  |
| /reload                                                | Reloads selected event                                                              |
| /roll                                                  | Sets playback mode to roll                                                          |

</TabItem>

| ------------ | ----------------------------------------------------- |

| /poll        | Returns the entire [runtime state](/api/runtime-data) |

</TabItem>
